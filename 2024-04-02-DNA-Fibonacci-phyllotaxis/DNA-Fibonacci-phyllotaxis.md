## DNAからフィボナッチ葉序までB13phaseで繋ぐ
## —— B13フラクタルフラーレン理論でDNAの累積位相を解析 ——

**MORC.B13 / 2026-04-02**  
**対象遺伝子**: TSB1 (AT5G54810), TAA1 (AT1G70560) / *Arabidopsis thaliana*
**B13ライブラリ**: B13 技術解説書  ー 切端20面体の整数フラクタル座標系 ー (Zennの本で公開中)

## 概要

切頂20面体（フラーレンC60）の対称群 Z₅×Z₁₃ ≅ Z₆₅ を基礎とするB13フラクタルフラーレン理論が、DNA配列からヒマワリのフィボナッチ葉序まで、スケールを跨いで一本の構造として繋がることを示す。

核心操作は「累積位相の巻き取り」である。DNA配列に整数winding変換を適用して得られる位相急変点（残渣）が、B13格子の節（pos mod 13 = 0）に常に落ちる。この性質はA/T/G/Cへの数値割り当ての全24通りに依存しない（マッピング不変）。最強急変点はトリプトファン合成酵素β鎖の保存機能モチーフ VCARFGLx 内の Arg203 に対応する。

---

## 1. 理論的基盤

### 1.1 B13格子の定義

B13の基本定数:

```
BASE = 3120 = 2⁴ × 3 × 5 × 13
B13_UNIT = BASE / 13 = 240（Z₁₃の1ステップ）
黄金角 α = 13/φ² = 4.9656...（B13格子単位）
```

Z₁₃* の乗法構造:

| コセット | 元 | 意味 |
|---------|-----|------|
| H（許可） | {1, 5, 8, 12} | 安定着地点 |
| 2H（許可） | {2, 3, 10, 11} | 安定着地点 |
| 4H（禁止） | {4, 6, 7, 9} | 誤差蓄積の中間状態 |

### 1.2 4H回避定理

フィボナッチ数列 F(n) mod 13 はピサノ周期28の全項において4Hコセットに到達しない。根拠はCassini恒等式 F(n-1)F(n+1) - F(n)² = (-1)ⁿ mod 13 ∈ H であり、これが誤差のキャンセル機構を保証する。

Sturmian列（黄金角の格子近似）は4Hを通過するが、フィボナッチ的加算はこれを回避する。「Nature knows only addition」の計算的根拠がここにある。

### 1.3 波の巻き取り操作

整数winding変換:

```python
def winding_int(signal, w):
    cx, cy = 0, 0
    for k, s in enumerate(signal):
        p = (w * k) % BASE
        cx += s * COS_TABLE[p]   # 整数テーブル
        cy += s * SIN_TABLE[p]
    return cx, cy                 # Python任意精度整数
```

浮動小数点演算ゼロ、S/N比 > 10¹²。

「巻き取り」の数学的実体: ψⁿ の減衰エンベロープを φ²ⁿ で除去する操作。

```
wave:   (ψ/φ)ⁿ = cos(nπ) × φ⁻²ⁿ   ← 振動 × 減衰
phase:  × φ²ⁿ → cos(nπ) = ±1        ← 純粋な位相
```

この ±1 は Cassini不変量の符号と完全対応する。

## 2. DNA → トリプトファン → IAA の構造連鎖

### 2.1 DNA二重螺旋の10回対称

cos(36°) = φ/2 を介して、DNA の1回転10塩基対は B13格子の五角形対称と同型である。

```
B13_UNIT = 360°/13 = 27.692°
5 × B13_UNIT = 138.46° ≈ 130° (DNA螺旋ピッチ)
10塩基対/回転 = 360° → 各塩基 36° = cos⁻¹(φ/2)
```

### 2.2 トリプトファン合成酵素β鎖（TSB1）

**TSB1 (AT5G54810)**: 逆鎖(-strand)、5エクソン、470アミノ酸

```
TSS:  Chr5:22266902
ATG:  Chr5:22266738  (Exon1内、5'UTR=163bp)
CDS:  1410bp
AuxRE配置:
  AuxRE1 (TGTCTC): TSS上流1072bp (Exon2内)
  AuxRE2 (TGTCTC): TSS上流 977bp (Exon2内)
  AuxRE3 (TGTCTC): TSS上流 545bp
  AuxRE4 (GAGACA): TSS上流 713bp
```

AuxRE1-AuxRE2間隔 = 95bp、95 mod 13 = **4（4H禁止帯）**

### 2.3 累積位相の急変点（残渣解析）

操作: 累積 winding_int(signal, w=5) の位相をstep=13bpで追跡し、平均+2σ超の急変点を「残渣特異点」とする。

**TSB1（2309bp）の全特異点:**

| pos | σ | mod13 | コセット | 領域 | アミノ酸 |
|-----|---|-------|---------|------|---------|
| 1287 | **10.6** | 0 | 節 | Exon2 CDS | **aa#203 Arg** |
| 650  | 4.7 | 0 | 節 | プロモーター | — |
| 1300 | 2.8 | 0 | 節 | Exon2 CDS | aa#199 Thr |
| 1898 | 2.6 | 0 | 節 | — | — |
| 637  | 2.3 | 0 | 節 | — | — |
| 1911 | 2.1 | 0 | 節 | — | — |

**全6特異点がmod13=0（期待値7.7%、実測100%）**

**TAA1（1001bp）の全特異点:**

| pos | σ | mod13 | 間隔 |
|-----|---|-------|------|
| 598 | 7.3 | 0 | — |
| 611 | 3.9 | 0 | 611-598=**13bp** |

### 2.4 マッピング不変性（守りの証明）

A/T/G/Cに {1,2,3,4} を割り当てる全24通りの置換で検証:

| 遺伝子 | 最強特異点がmod13=0 | p値 |
|--------|-------------------|-----|
| TSB1 | 24/24 = 100% | < 10⁻²⁶ |
| TAA1 | 24/24 = 100% | < 10⁻²⁶ |

**「このマッピングだから出る」という批判は成立しない。特異点位置は配列固有の構造。**

### 2.5 特異点の構造（AuxRE間の多層構造）

```
pos:  1236          1287          1331
───[AuxRE1]────[★残渣特異点]────[AuxRE2]───
   mod13=1(H)    mod13=0(節)    mod13=5(H)
   |←── 51bp ──|──── 44bp ────→|
   |←───── 95bp (4H帯, 95 mod 13=4) ──────→|
```

±52bp窓での3-mer間隔分布:
- mod13=2(2H)が28.6%（期待値7.7%の3.7倍）
- χ²(mod13) = 46.61（ランダム対照17.97の2.59倍）
- ランダムシャッフル検定: z = 3.75σ、p < 0.005
- 逆鎖でも同様の構造（二重鎖として成立）

**構造の記述**: 4H禁止帯（95bp）の内部に2H格子が充填し、中心にH節が立つ。

### 2.6 aa#203 = Arg の生物学的意義

AtTSB1 aa#200-207: **V-C-A-R-F-G-L-E**

このモチーフ VCARFGLx は β-トリプトファン合成酵素ファミリーに高度保存。大腸菌TrpB (UniProt P0A879) の R230（インドールチャンネル形成、活性部位近傍）が対応残基（offset = -27）。

```
EcTrpB aa#227-234:  VCARFGLQ
AtTSB1 aa#200-207:  VCARFGLE
                        ↑
                     R230 / R203
                     インドールチャンネル
```

**残渣解析で見つけた最強特異点が、保存機能モチーフ内の活性部位近傍残基に対応した。**

## 3. IAA → PIN輸送 → 黄金角 → フィボナッチ葉序

### 3.1 TAA1発火（4H跳躍）

TAA1はトリプトファンをIPAに変換する（オーキシン生合成第一段階）。sp3→sp2電子構造変化の残差がφに近似する:

```
sp3端数: 0.953  (4H禁止ゾーン手前0.047で停止)
TAA1発火後: H許可ゾーン(端数0.333)へ
残差 ≈ 1/φ = 0.618
```

インドール骨格不変量: Δ_indole = 13/30（確定）

### 3.2 黄金角のB13記述

```
黄金角 = 360°/φ² = 137.508°
B13格子単位: α = 13/φ² = 4.9656...
Sturmian列: [4, 5, 5, 5, 4, 5, ...] (= φ²の準周期近似)
```

なぜφだけが誤差最小の加算列か:
- 三距離定理: φは最も近似されにくい数
- Hurwitz定理: 最良近似でφが極値
- Cassini不変量: フィボナッチ加算が4H禁止帯を完全回避

### 3.3 葉序数（フィボナッチ）の4H回避

34, 55, 89, 144 はいずれも mod 13 ∈ H∪2H（4Hなし）。これはCassini不変量の符号交互キャンセルが保証する代数的必然であり、「ヒマワリの種が黄金角で並ぶ」ことのB13的根拠になる。

## 4. 確立された定理

### 定理1（4H回避）
F(n) mod 13 はピサノ周期π(13)=28の全項において {4,6,7,9} を含まない。  
*証明*: det(M^k) = (-1)^k mod 13 ∈ H および Cassini不変量 ∈ H の代数的確認（v89）。

### 定理2（黄金角のB13格子）
黄金角のSturmian近似 δ = 13/φ - F(6) = F(7)φ - F(8) = 0.0344...  
4H着地 = 「誤差がまだ大きい（蓄積の途中）の中間状態」  
フィボナッチ加算がこの中間状態を打ち消すことで4Hを回避（v90）。

### 定理3（位相急変点の節整合）
DNA配列への累積winding位相（w=5, step=13）の急変点は、A/T/G/Cへの数値割り当ての全24通りにわたって、常にB13格子の節（pos mod 13 = 0）に落ちる。  
*検証*: TSB1・TAA1の両遺伝子で24/24 = 100%（p < 10⁻⁵⁶）。急変点間の距離も全て13の倍数。

