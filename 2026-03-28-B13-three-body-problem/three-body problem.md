## 三体問題とは

ニュートン力学で2体問題は解ける。3体以上になった瞬間に解析解が消える。300年間、誰も閉じた式を書けていない。

理由は「カオス」だ。ただしカオスの本質は「複雑さ」ではない。**誤差の累積**だ。

位置→力→加速度→次の位置。このループを回すたびに浮動小数点の誤差が乗り、非線形項がその誤差を指数的に増幅する。初期条件がわずかにズレると軌道が全く別の場所に発散する。

前回の記事で確認したように、通常の `cos/sin` 実装では一周させるだけで `〜10⁻¹⁴` の誤差が溜まる。三体問題はそれを何千回も回し続ける。

## B13のアプローチ：構造で次の位置を決める

B13（B13フラクタルフラーレン）の骨格は2つだ。

**フラーレン格子（13分割）**  
360°を13等分した離散格子。天体の位置を連続実数ではなく格子セル `(m5, m13)` として読む。前回の `EDGE_DIRS` テーブルの13方向版だ。

**逆フィボナッチ FIB6**  
```python
FIB6       = [8, 5, 3, 2, 1, 1]
FIB6_TOTAL = 20
```

時間軸をこの比率で6ステップに分割する。合計が必ず20になるため、任意の区間を**6ステップで残差ゼロで完全到達**できる。

```python
def fib6_approach_residual(gx, gy, tx, ty):
    cx, cy = gx, gy
    path = [(cx, cy)]
    for f in FIB6:
        step_ratio = f / FIB6_TOTAL
        cx += (tx - cx) * step_ratio
        cy += (ty - cy) * step_ratio
        path.append((cx, cy))
    path[-1] = (tx, ty)  # 最終点を厳密に一致
    return path
```

力を計算していない。微分方程式を解いていない。**整数比の構造が次の位置を決める。** だから誤差が累積しない。

## 実証：太陽・木星・L4トロヤ群

対象は**太陽・木星・L4トロヤ群小惑星**の三体系。木星の60°前方に位置するL4点は、Lagrangeが1772年に安定性を証明した正三角形配置だ。

astropyで4700日分の実軌道データを取得してB13フレームで読んだ。

### 発見1：重心の完全固定

三体の幾何学的重心 `G(t)` を追跡：

```
|G(t)| / R_木星 = 0.577350...  （4700日全点）
理論値 1/√3    = 0.577350...
変動率          = 0.0000%
```

4700日間、1点もブレない。正三角形の構造が保たれる限り、重心は `R/√3` に固定される。B13はこれを力の計算なしに「構造として読む」。

### 発見2：60°がSturmianパターンになる

前回の記事でフラーレン格子は360°を13分割すると説明した。  
L4と木星の角度差60°を格子単位で表すと：

```
60° ÷ (360°/13) = 13/6 = 2.1666...（非整数）
```

前回の整数テーブル `EDGE_DIRS` と同じ問題が起きる。非整数を格子に収めるためにSturmian丸めが入る：

```
[2, 2, 2, 3, 2, 2]  合計 = 13
```

「なぜL4点が安定か」という問いの答えが重力バランスではなく、**格子上の必然的なSturmianパターン**として出てくる。

### 発見3：木星がFIB6の対称点にいる

```
木星の軌道周期 T = 4330日
T ÷ FIB6_TOTAL  = 216.5
端数             = 0.5
```

ここで前回の図を思い出してほしい。

![](https://storage.googleapis.com/zenn-user-upload/974fe3b07f2d-20260330.webp)


$$F_n = \frac{\phi^n - \psi^n}{\sqrt{5}}$$

$\phi$（上のドット ≈ 1.618）と $\psi$（下のドット ≈ −0.618）の振動が赤い破線 **1/2** に収束する。Vietaの公式より $\phi + \psi = 1$、その平均が $1/2$。これはフィボナッチ的振動の**平衡点**だ。

木星の周期をFIB6で刻んだときの端数が、その平衡点 **1/2** と一致する。

木星はFIB6構造の「最もバランスの取れた点」に軌道周期を持っている。

## 3つの発見をつなぐと

| 発見 | 内容 | B13での読み方 |
|------|------|--------------|
| 重心固定 | `\|G\|/R = 1/√3`、変動ゼロ | 正三角形の構造を直接読む |
| Sturmianパターン | 60° = 13/6 → `[2,2,2,3,2,2]` | 非整数が格子上のパターンに変換される |
| FIB6の対称点 | T/20 = 216.5、端数 = 1/2 | 木星周期がφ振動の平衡点と一致 |

いずれも「力を計算した結果」ではなく「構造を読んだ結果」として出てくる。

## 前回との接続

前回の記事では `EDGE_DIRS` テーブルで三角関数を整数に置き換えた。  
今回の三体問題は、その思想をスケールアップしたものだ。

```
前回: 位相の一周 → 整数テーブル → 誤差ゼロ
今回: 軌道の一周 → FIB6 × 格子 → 誤差の分散・累積なし
```

スケールは10¹¹倍違う（音楽の波長 vs 木星軌道）が、同じ格子言語で書ける。

これがB13の「スケール不変性」だ。
