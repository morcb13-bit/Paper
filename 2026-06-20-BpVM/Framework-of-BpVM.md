## 0. この記事が置きたい一文

最後に、たった一文を正確に置きたい。

> **特別なものは何もない。**

これは標語ではない。仮定でもない。**結論**として置きたい。そのために、巻取り機を「真に万能」にしようとして突き当たった壁を、順番に越えていく。越え終わったとき、この一文がなぜ真なのかが、言葉になって残るはずだ。

## 1. 問い ── 揺れを追えない機械は、何も追えていない

B13 の巻取り機は、信号を単位円に巻き付け、巻き付け周波数が成分と一致したときに重心が跳ねることで、その成分を読む。一致したら符号を反転して引き算し、残りを探す。玉ねぎを剝くように、大きな波から順に剝いでいく道具だ。

この道具は、整数比の調和構造はきれいに剝ぐ。だが長く壁になっていたものがある。**ビブラート**だ。

ビブラートは特別な技巧ではない。手の動き、運指、口唇、息継ぎ──生身の演奏が無意識に乗せる微小な揺れであり、ストレートトーンのつもりの音にも入っている。つまり実演奏の搬送波は、最初から一つの周波数に座っていない。

ここに気づくと、問いの形が変わる。「ビブラートだから追えない」は、「**生きた信号の搬送波は一つも追えていない**」と同義だ。止まった鋳型（定常な正弦波）で生身の音を剝げば、特定の音だけでなく、あらゆる音が残渣を残す。これは特殊例のバグではなく、定常基底と生きた音源の、一般的なミスマッチだった。

そして揺れは、音楽だけの話ではない。風に揺れるアンテナの信号は風で変調され、伴星を連れた連星の周期は伴星に揺すられる。**自然界に、揺れはつきものだ。** 揺れを扱えない機械は、理想試験管の中だけで動く機械であり、それは「ブラウン運動は確定値だ」と言い張るのと同じ嘘をつくことになる。

## 2. 揺れは雑音ではない ── 一つ上の搬送波として読む

修理の鍵は、鋳型を止めていたこと自体にあった。**鋳型（atom）は点ではなく、軌道でなければならない。**

ビブラート音は周波数変調（FM）だ。搬送波の周波数 f₀(t) が、ゆっくりした揺れの上に乗っている。手の揺れはおおよそ周期的で、毎秒数回。だから揺れは「読めない連続」ではなく、**一つ遅い搬送波**として立ち上がる。

ここから機械は塔になる。

```
階層0(速) : 搬送波帯を巻く → 各フレームの重心 → f₀(t) の軌道
階層1(遅) : f₀(t) の軌道を巻く → ビブラート速度・深さが共鳴として立つ
階層2(更遅): 運弓・息のフレーズ包絡が立つ
   ⋮
```

ビブラートを「剝がす特殊な部品」ではなく、「一つ上の階層の、普通の搬送波」として、**同じ巻取り機を再帰適用する**。柔らかさは、鋳型を自由に変形させることからではなく、巻取りの階層を積むことから来る。各階層は相変わらず厳密な加算による巻取りのままだ。揺れに合わせて機械を曲げるのではなく、揺れを次の階層の信号として迎える。これが「しなやかさ」の正体だった。

連星の周期の揺れを読むことが、伴星を読むことであるように。実際、最初に確認された系外惑星は、ミリ秒パルサー PSR B1257+12 のパルス到来時刻の揺れ──周期のわずかなずれ──を読んで見つかった（Wolszczan & Frail, 1992）。揺れはノイズではなく、**揺れこそが発見**だったのだ。

## 3. 構造の有無を返す ── null という正直さ

塔を上に伸ばしていくと、揺れには二種類あることが分かる。

- **(a) 構造変調**：ビブラート、乱流のカスケード、伴星の軌道。特性スケールを持つ。塔が階層を登って読む。
- **(b) スケール自由なゆらぎ**：ブラウン運動、熱雑音、量子的なゆらぎ。どの階層にも搬送波がない。

ここで、この機械の最も大事な性格が出る。**この機械が各階層で返すのは「値」ではなく「構造の有無」だ。**

普通の信号処理や、いまの多くの推論機械は、入力に対して**必ず何かを返す**。値を返さないことが、まるで失敗であるかのように振る舞う。BpVM は違う。構造があれば読み、構造がなければ「ここに構造はない（null）」と返す。

この null には、ひとつ反転がある。塔が (b) に返す null は、「ブラウン運動を確定値だ」と言っているのでは**ない**。むしろ正反対だ。誤りとは、ゆらぎに無理やり綺麗な確定値を押し付けること。null はその逆をやる──**構造のない所に構造を捏造しない**。「ここに算術構造はない」と返すのは、それが還元不能なゆらぎだという正直な報告であって、確定値の主張ではない。偽の値を吐く機械の方が嘘つきで、null を返す機械の方が正直なのだ。

風のアンテナは両方を含んでいて美しい。乱流には構造化されたカスケードがあり、塔はそれを登って読む。そして最小スケールで熱散逸＝真のゆらぎに底を打つ。**塔が登りきって「構造なし」を返す場所が、構造の尽きる場所の測定になる。**

## 4. 無限には剝かない ── 「1」とキャリー

剝くのは無限ではない。**必要精度の「1」を基準に置き、それ以下は残渣としてキャリーする。** 深さは課題が要求する精度で止まる。

ここで決定的なのは、**キャリーは切り捨てではない**ことだ。

```
切り捨て : 1以下を 0 にする → 「そこは綺麗だ」と嘘をつく
キャリー : 1以下を残渣として持ち越す → 「今の分解能の下にある」と正直に言う
```

機械が不正直になるのは、残渣を 0 に丸めて「ゆらぎは無かった」と振る舞う瞬間だ。キャリーはそれをしない。残渣を**信号とも断じず、捨てもせず、「分解能の下」として勘定に残す**。だからゆらぎを解像しないまま、ゆらぎを偽装もしない。塔が底まで無限に剝かずに済むのも、ブラウン運動を確定値と呼ばずに済むのも、この一点のおかげだ。

## 5. 「1」とは何か ── プローブが決める床

では「1」を、究極には何が決めるのか。実務的には計算コストと必要精度で自由に置けばいい。だが越えられない床は、観測の手段が決めている。

**究極の床がどこにあるかは、理論に依存する。** ここではそこに踏み込まない。代わりに、こう定義する──**「1」とは、観測プローブが決める最小分解能である。**

私たちはふだん、電磁波で世界を調べる。光を光で測る以上、ある分解能より細かいものは、光では区別がつかなくなる。そして器が宇宙と同じ素材でできているがゆえに、測定の限界そのものが一つの自然現象として立ち現れる。「1」は、宇宙の中の現象であると同時に、物差しの限界でもある──この二面は、本当は一つのものだ。

ただし、この「1」は**観測手段に相対的**だ。プローブを変えれば──重力波でも、ニュートリノでも──「1」は動く。だから「1」より下を「構造がない」と読むのは禁物で、正しくは「**このプローブでは解像できない**」。ここでキャリーが効く。残渣を捨てれば「この先には何もない」と嘘になるが、キャリーすれば「別のプローブなら解けるかもしれない」を正直に残す。

電磁波で言えば、宇宙には**それより深部を光では直接観測できない面**がある（宇宙背景放射の、最後の散乱面）。その向こうは、光にとっては残渣だ。だがそれは「向こうに何もない」ことを意味しない。**ただ、この光では届かない。** 「1」とは、そういう床のことだ。

## 6. 周波数も、スケールも、次元も、同じ一つの加算

ここまで、塔は二つの方向に伸びてきた。**スケール**（剝く深さ）と**周波数**（巻き付け）だ。残る方向がある。**次元**だ。

フィボナッチ螺旋を立体で読むと、(0,0,0) という節点で全軸が同時に反転し、1/13 スケールで自己相似に再出発する。ここで決定的なのは、この反転と 1/13 が **次元数に依らない**ことだ。x 軸でも、z 軸でも、n 番目の軸でも、同じ漸化式 F(n+1)=F(n)+F(n−1) が、同じ節点で、同じ反転と同じリスケールをやる。次元を一つ増やすことは、新しい仕組みを足すことではない。**同じ一個の操作を、もう一本の軸に向けて回すだけ**だ。

ここで、三つの方向が一点に潰れる。

- **周波数**を巻くこと ＝ 位相の加算
- **スケール**を降りること ＝ 節点での加算（自己相似の再帰）
- **次元**を増やすこと ＝ 軸方向のインデックスの加算

これらは三つの異なる操作ではない。**ただ一つの加算を、違う軸に向けたものだ。** 周波数・スケール・次元という、計算機なら普通は別々のサブシステムで扱う三つが、ここでは同じ命令の三つの向きに過ぎなくなる。

そして、この潰れこそが、機械の名前を決める。

操作がこれほど軸に依らないなら、それはもう、特定の素数 13 の固有構造にも依らない。同じ命令は、どの素数 p でも同じように立つ。p は「どの位 Z/p を読むか」「どの漸化式を回すか」を選ぶ**ダイヤル**になり、機械の骨格は p をまたいで不変になる。**軸に依らないことが、素数に依らないことを露わにする。**

B13 という一つの実例を、Bp という一つのアーキテクチャへ持ち上げるのは、この瞬間だ。p を回せば空間の自由度は増えるが、回しているのは同じ機械である。だからこれは B13VM ではなく、**BpVM** なのだ。

念のために言えば、ここで主張しているのは数学の定理ではない。**一つの機械を、どう設計するか**だ。三つの軸が一つの加算に潰れる——その設計思想の上に、この機械は立っている。

## 7. BpVM（フラクタル仮想マシン）構想 ── 軸に依らない一つの加算

ここまで来ると、機械の仕様が、曖昧な美点ではなく具体的な形になる。

```
命令セット : ただ一つ ── フィボナッチ加算 ＋ 節点での全軸反転・1/13 リスケール
軸         : 次元 / スケール / 周波数 / 時間 ── どれでも、何本でも自由
出力       : 値ではなく「構造の有無」── 在れば読み、無ければ null とキャリー
床 (1)     : 観測プローブと cost/precision で自由に置く
天井       : 13^52(B13 実例の、不変の枠)
ダイヤル   : 素数 p(B13 を Bp へ持ち上げる)
```

柔軟なのは、命令が**軸に依らない一個**だからだ。軸を足すのは新機能の追加ではなく、同じ一個を一本多く回すだけ。機械は、加算が残した剰余を読み、構造の有無を返す。

これが **"Nature knows only addition"** の、機械としての姿である。

---

## 8. 「特別なものは何もない」が、真である理由

そして、置きたかった一文に戻る。

枠の中で、本当に特別だったものが二つある。**重なった源を分けるためのプライア**（学習済みの事前知識）と、**「1」の下にあるスケール自由なゆらぎ**だ。塔はこの二つを中に抱え込まない。前者は「分離」の境界で外へ手渡し、後者は「1」の床でキャリーとして外へ送る。

つまり、機械の中身が一様であるのは、**不均一なものをすべて縁に逃がしてあるから**だ。

だから「特別なものは何もない」は、楽観ではない。それは**二つの境界を引いた代償として手に入った性質**である。中が一つの加算だけで済むのは、加算で済まないものを縁に出し切ったから。ここを忘れて「この機械が全部やる」と読めば、プライアとゆらぎが中へ戻ってくる。一様さは、境界の正直さに支えられている。

---

## 9. 結 ── 旅が、装置だった

ビブラート → 塔 → キャリー → 「1」→ プローブの光 → 次元 → BpVM。

しなやかな BpVM とは、結局のところ、**軸に依らないフィボナッチ加算を一個、好きな軸に好きなだけ回し、プローブで床を、13^52 で天井を持ち、値ではなく構造の有無を返し、二つの縁で正直に手渡す**機械にすぎない。「すぎない」と言い切れるのは、何が中で何が縁か、もう全部が言葉になっているからだ。

旅そのものが、あの一文を仮定から結論へ移すための装置だった。

特別なものは、何もない。
だから──

> **Nature knows only addition.**
