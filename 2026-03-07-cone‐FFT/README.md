## これは何か？

cone-FFT は、2つの独立した層から構成される信号変換である。

1. **Parseval 本体** — ノルムスケーリング係数が厳密に40となる2チャネル整数再帰
2. **コーン幾何** — (3,4) 回転軌道から導かれる局所幾何不変量。黄金比 φ を**仮定でなく結果として**導出する

この2層は、因子 **5** という構造的接点を1つだけ共有する。

## 核心結果

### 再帰行列

```python
M = [[6, 2],
     [2,-6]]
```

は **M² = 40I** を満たす。これにより1段ごとに厳密な整数ノルムスケーリングが保証される。

```python
from parseval_core import step_rule_40, step_rule_40_inv

# 順変換: ノルム ×40
T1, U1 = step_rule_40(T0, U0)

# 逆変換: ノルム ÷40（整数除算で厳密）
T0r, U0r = step_rule_40_inv(T1, U1)
assert T0r == T0  # 完全復元
```

### コーン連鎖

(3,4) の4点軌道から：

```
G = ((25,-25),(-25,50))  →  判別式 = 5⁵  →  ρ_eig = φ²  →  ρ_a = √5
→  R = -φ⁻²  →  C_CONE = 2/φ
```

φ は導出される。仮定ではない。

```python
from cone_local import cone_chain
ch = cone_chain()
# ch['R']      == -0.38196601  (-φ⁻²)
# ch['C_CONE'] ==  1.23606798  (2/φ)
```

## BASE = 3120

```
3120 = lcm(3, 16, 5, 13)

因子 | 意味
-----|-----
3    | 平衡3進の底
16   | r_local 符号周期
5    | √5 幾何（φへの入口）
13   | B13 格子
```

B13 × √5 が共存できる平衡3進の最小深さ（深さ4）から定まる最小公倍数。
