# cone-FFT: 局所コーン幾何を持つ整数2チャネル変換

## 概要

cone-FFT は、2つの独立した層から構成される整数信号変換である。
第1層は、ノルムスケーリング係数が厳密に40となる2チャネル整数再帰。
第2層は、(3,4)回転系の4点閉軌道から導かれる局所幾何不変量。

本稿では、再帰行列が M² = 40I を満たすことを証明し、
整数上での完全な可逆性を確立する。
また黄金比 φ が仮定ではなく、局所 Gram 行列の判別式構造の
**結果として**出現することを示す。

**完成している部分：** 命題A–D、整数順/逆変換、cos+sin Parseval 等式、BASE=3120 の構成。  
**未完成の部分：** エンドツーエンドの整数FFT系を閉じる、cos+sin 二チャネル完全可逆パイプライン。

## 1. 背景と動機

### 1.1 田問題

出発点は Z² 上の4点閉軌道である：

```
中心: (7, 7)
基底: (3, 4)、  3² + 4² = 25 = 5²

4点軌道（±90°回転）:
  (7,7) + ( 3, 4) = (10, 11)
  (7,7) + ( 4,-3) = (11,  4)
  (7,7) + (-3,-4) = ( 4,  3)
  (7,7) + (-4, 3) = ( 3, 10)

全座標値: {7±3, 7±4} = {3, 4, 10, 11}
```

**核心的観察：** 全ての座標は整数の加法 7±3 または 7±4 のみ。
この定義には黄金比 φ はどこにも現れない。

### 1.2 BASE = 3120

整数底は以下のように構成される：

```
Step 1: 平衡3進（底3）を選ぶ
Step 2: B13 × √5 が共存できる最小深さ = 4
        深さ3: repunit = 13（B13の核、因子5なし）
        深さ4: repunit = 40 = 8×5（初めて因子5を含む）
Step 3: 3⁴ - 1 = 80 = 2⁴ × 5  →  因子16, 5
Step 4: B13  →  因子13
Step 5: BASE = lcm(3, 16, 5, 13) = 3120
```

## 2. 二層構造

cone-FFT は2つの独立した層に分離される：

| 層 | 内容 | 起源 |
|---|---|---|
| **Parseval 本体** | 整数2チャネル再帰、ノルム×40 | 代数的 |
| **コーン幾何** | 局所 Gram → φ² → √5 → −φ⁻² → 2/φ | 幾何的 |

この2層は構造的接点をただ1つ共有する：因子 **5**
（Parseval 層では 40 = 8×5、コーン層では判別式 = 5⁵）。

## 3. 命題

### 命題A — Parseval ノルムスケーリング

**主張.** 整数2チャネル再帰

```
T' = 6T + 2U
U' = 2T − 6U
```

は任意の整数ベクトル T, U に対して厳密に以下を満たす：

```
‖T'‖² + ‖U'‖² = 40 (‖T‖² + ‖U‖²)
```

**証明（代数的）.** M = [[6,2],[2,−6]] とおく。直接計算により：

```
M² = [[6,2],[2,−6]]² = [[40,0],[0,40]] = 40I
```

よって：
- B 偶数：M^B = 40^(B/2) I、ノルム係数 = 40^B
- B 奇数：M^B = 40^((B−1)/2) M、ノルム係数 = 40^(B−1) × 40 = 40^B  □

### 命題B — コーン局所固有値連鎖

**主張.** V = {(3,4),(4,−3),(−3,−4),(−4,3)} を4点軌道の偏差ベクトル、
W = {V[(i+1)%4] − V[i]} をその差分ベクトルとする。
(V[0], W[0]) の局所 Gram 行列は：

```
G = ((25, −25), (−25, 50))
```

これより：

```
判別式  = trace² − 4·det = 75² − 4·625 = 3125 = 5⁵
√判別式 = 25√5
ρ_eig  = √(λ₊/λ₋) = φ²
ρ_a    = ρ_eig − 1/ρ_eig = φ² − φ⁻² = √5
R      = (1 − ρ_a)/(1 + ρ_a) = (1−√5)/(1+√5) = −φ⁻²
C_CONE = 2 + 2R = 2/φ
```

**系.** φ は仮定ではない。(3,4) 回転軌道の整数 Gram 構造の結果である。

### 命題C — 整数可逆性

**主張.** 再帰は整数上で厳密に可逆である：

```
T = (6T' + 2U') / 40
U = (2T' − 6U') / 40
```

**証明.**

```
6T' + 2U' = 6(6T+2U) + 2(2T−6U) = 40T
2T' − 6U' = 2(6T+2U) − 6(2T−6U) = 40U
```

よって40による除算は任意の整数入力に対して厳密に割り切れる。
B段逆変換も帰納法により同様。  □

### 命題D — 40 の起源

**主張.** スケーリング係数40は、4つの独立した条件によって同時に決定される：

```
(1) 代数:   6² + 2² = 40  が a²+b²=40 の唯一の整数解
(2) 行列:   M² = 40I  (M は 40I の整数平方根)
(3) 数論:   (3⁴−1)/2 = 40  (3進 repunit 深さ4)
(4) 共有因子: 40 = 8×5、判別式 = 5⁵
             因子5 が両層に現れる
```

## 4. cos+sin Parseval 等式

N = BASE = 3120 に対して：

```
cos_proj[k] = Σⱼ f[j] · round(BASE · cos(2π jk / BASE))
sin_proj[k] = Σⱼ f[j] · round(BASE · sin(2π jk / BASE))
```

**実測結果：**

```
‖cos_proj‖² + ‖sin_proj‖² ≈ BASE² × N × ‖f‖²   （比 ≈ 1.0）
```

これは標準DFT Parseval 等式の整数スケール版である。


これが現状と完全な整数FFT系の間にある唯一のギャップである。
