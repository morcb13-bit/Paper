## 中心命題

40×40×40 格子上の Bloch 型スキルミオンチューブに対し、coneFFT 由来の指標 Φ_d は
一様場では厳密に 0、スキルミオン場では非ゼロとなり、両者の比は約 10⁹ 倍に達した。
これは、通常 FFT が直接は表現しにくい回転位相構造を、R₆ 型の向き付き相関が
選択的に捕捉している可能性を示す第一観測である。

## §1 動機と位置づけ

本ノートは、完成理論の主張ではなく、**観察と検証のための最初の記録**である。
coneFFT は「FFT がレントゲンなら、coneFFT は MRI」という比喩のもと、
R₆ フィボナッチ差分演算子を核心に据えた信号変換の研究構想である（v36 引継書参照）。

この段階で問うべき問いはシンプルである：

> **観察問**：coneFFT 由来の指標 Φ_d は、渦構造（スキルミオン）に対して通常場と異なる応答を示すか？

複雑な実在材料・厳密エネルギー最小化・実験定量再現は、現段階の目標ではない。
人工的な解析解スキルミオン場を用い、**対照群（一様場）との比較**によって
指標の識別能力を確認することが第一目標である。

## §2 モデル定義

### 2.1　40³ 格子とスキルミオンチューブ

40×40×40 の格子（64,000 セル）を用いる。各セルに単位スピンベクトル
**S**(x,y,z) = (Sx, Sy, Sz) を割り当てる。
スキルミオンチューブは z 軸方向に貫通する Bloch 型として、以下の解析解で定義する：

```
r = √((x−x₀)² + (y−y₀)²)      // 中心からの半径距離
θ(r) = 2 arctan(R / r)          // 極角：スキルミオン半径 R=0.35
φ(x,y,z) = arctan(y/x) + λz    // 方位角 + z方向ねじれ λ=0.5π

S = ( sin θ cos φ,  sin θ sin φ,  cos θ )   // 正規化済み
```

一様場（対照群）は S = (0, 0, 1) の定数場として定義する。

> **モデルの性格**：本解析解は数学的に都合のよい人工モデルであり、特定材料の LLG 方程式解ではない。スキルミオン形状の「骨格」を確認するための観察モデルとして位置づける。

### 2.2　R₆ 演算子と Φ_d 不変量

coneFFT の核心演算子を以下に定義する（v36 実装と同一）：

```
R₆ フィボナッチ差分演算子（循環境界条件）

R₆(x)[p] = x[p+1]−x[p−1]
           + x[p+2]−x[p+3]
           + x[p+5]−x[p−8]    // インデックスは mod n

双対ノード
N(a, b) = (a+b, a−b)           // 加算・差分の対

層不変量 Φ_d
Φ_d(x) = (1/n) Σ_p R₆(x)[p] · R₆(x)[p+2]
```

本実験では 40×40×40 の各 z 断面について、中央 y ラインの Sz 成分を
1D 信号として R₆ と Φ_d を計算した。

## §3 数値実験の結果

### 3.1　XY / XZ 断面の可視化

図は HTML ビジュアライザー（skyrmion-coneFFT.html）による断面図に対応する。
XY 断面（z=20）では Sz が中心で −1 付近、外縁で +1 へと連続的に変化し、
Bloch 型スキルミオン渦の方位角回転が確認できる。
XZ 断面ではチューブが z 軸方向に貫通する様子が視認され、λ=0.5π のねじれ成分も
Sx/Sy カラーマップに周期的パターンとして現れた。
一様場断面は全セルで Sz=1 の単色となり、対照群としての均質性を確認した。

> **可視化確認事項**：iso_points（Sz ≈ −0.5 の領域）は 1,120 点検出。スキルミオンコア領域が格子全体 64,000 セルの約 1.75% を占めることを定量的に確認した。

### 3.2　Φ_d の z 依存性

各 z 層（0〜39）について、中央 y ラインの Sz 成分を 1D 信号として Φ_d を計算した。

| 場の種類 | Φ_d（全層平均） | z 方向の変動 |
|---------|----------------|------------|
| スキルミオン場 | 1.1308 | z に依存して緩やかに変動 |
| 一様場 | ≈ 0（丸め誤差 ~10⁻¹²） | 全層で厳密にゼロ |
| **比（SKY / UNI）** | **≈ 10⁹ 倍** | — |

この桁差は単純な振幅差ではない可能性がある。一様場では S(**r**) = const であるため、
R₆ 演算子の正負の寄与が完全にキャンセルし、Φ_d は**対称性によって消滅**する。
一方スキルミオン場では spin orientation winding により回転成分が空間的に固定されており、
R₆ 型の向き付き演算子をかけると寄与が同符号で積み上がる。

> **構造的桁差の解釈（仮説）**  
> Φ_d（一様）= 厳密 0（対称性消滅）  
> Φ_d（スキルミオン）= O(1)（位相巻き付きによる残存）  
> → 比 ≈ 10⁹ は振幅差ではなく「ゼロ ↔ 非ゼロ」の構造差である可能性

### 3.3　トポロジカル電荷 Q(z)

各 z 層のトポロジカル電荷 Q(z) を離散 skyrmion number 積分で評価した：

```
Q(z) = (1/4π) Σ_{x,y} S · (∂xS × ∂yS) Δx Δy
```

| 指標 | 値 |
|------|-----|
| Q(z) 平均（各層） | ≈ −0.90 |
| Q 合計（全 40 層） | ≈ −36 |
| 一様場 Q | ≈ 0 |

Q ≈ −1/層 はスキルミオン 1 個（巻き数 −1）の理論値と定性的に整合する。
全層合計 Q ≈ −36 は 40 層のチューブ積分に対応するが、
**連続体の厳密整数不変量ではない**ことに注意が必要である（§5 参照）。
反スキルミオン実験（φ 反転モデル）では Q ≈ +34.2 と符号反転を確認し、
指標の符号感度を検証した。

### 3.4　FFT との比較

中央 y ライン Sz 信号について、通常 FFT パワースペクトルと
coneFFT 由来の Φ_d を比較した。

| 指標 | スキルミオン | 一様場 |
|------|------------|--------|
| FFT peak（k=1–5） | 4.562 | 0.0 |
| Φ_d（平均） | 1.1308 | ≈ 0 |

FFT もスキルミオン場に対して非ゼロのピークを示すが、一様場（定数信号）では
直流成分（k=0）のみに電力が集中するため k≥1 の帯域ではゼロとなる。
この意味では両指標とも渦構造を検出できる。
ただし FFT の検出は**空間変調の有無**（振幅的差異）に基づくのに対し、
Φ_d は**向き付き相関の有無**（対称性的差異）に基づく可能性があり、
両者が捕捉する情報の種類は異なる可能性がある。この点は今後の解析課題である。

### 3.5　拡張実験：半径・多体・反スキルミオン

3 種の変形モデルについて同様の計算を実施した：

| 実験 | 内容 | Φ_d（平均） | Q（平均/層） |
|------|------|------------|------------|
| R=0.15 | スキルミオン半径縮小 | 0.1062 | −0.865 |
| R=0.25 | 標準より小さい半径 | 0.1370 | −0.881 |
| R=0.45 | 標準より大きい半径 | 0.1616 | −0.814 |
| 2-sky | 2 スキルミオン重ね合わせ | 0.1447 | −0.688 |
| 反スキルミオン | φ 符号反転 | 0.1535 | **+0.855** |

半径増大に伴い Φ_d が単調増加する傾向が見られ、コア体積と Φ_d の相関が示唆される。
反スキルミオンでは Q の符号反転（負 → 正）が明確に確認され、
指標がトポロジカル巻き数の符号を識別できることを示した。

## §4 観察のまとめ

本数値実験から得られた観察事項を以下に整理する：

1. **Φ_d の識別能力**：一様場では厳密にゼロ、スキルミオン場では非ゼロ。その比は約 10⁹ 倍であり、coneFFT 指標が回転位相構造を選択的に捕捉している可能性の第一観測が得られた。

2. **Q との整合性**：Φ_d 非ゼロの場でトポロジカル電荷 Q ≈ −0.90/層が確認され、理論値（−1）と定性的に整合した。

3. **符号感度**：反スキルミオン（φ 反転）において Q の符号が反転することを確認した。

4. **半径依存性**：スキルミオン半径の増大に伴い Φ_d が増加する傾向が観察された。

5. **FFT との差異**：両指標ともに渦構造を検出するが、Φ_d は向き付き相関という FFT とは異なる側面を捕捉している可能性がある。

> **第一観測の結論**  
> 40³ 格子 Bloch 型スキルミオンチューブに対し、coneFFT 由来の指標 Φ_d は
> 一様場とは構造的に異なる応答を示した。
> これは coneFFT が渦位相構造のトポロジカル検出器として機能する可能性を示す
> **最初の数値的証拠**である。

## §5 誠実な限界の明記

本観察には以下の重要な限界がある。研究ノートとして誠実に記録する：

> **限界①：Q は厳密な整数不変量ではない**  
> Q ≈ −36 は連続体位相幾何学の厳密整数不変量ではない。有限格子サイズ・有限スキルミオン半径・周期境界条件・離散微分スキームのすべてが影響する離散評価値である。

> **限界②：人工解析解モデル**  
> 使用したスキルミオン場は LLG 方程式のエネルギー最小化解ではなく、数学的に都合のよい解析解（θ(r) = 2 arctan(R/r)）である。実在材料の磁気構造とは直接対応しない。

> **限界③：Φ_d の 1D 実装**  
> 本実験の Φ_d は各 z 層の中央 y ラインを 1D 信号として計算した。R₆ 本体が 3D 格子の対角方向相関を想定するならば、この 1D 実装は本来の演算子能力を部分的にしか実現できていない可能性がある。

> **限界④：Φ_d の理論的正当化は未完**  
> Φ_d が orientation-sensitive な量であるという解釈は現時点では観察からの推論であり、解析的な証明は行われていない。10⁹ 倍差の構造的起源については追加検証が必要である。

## §6 次のステップ

本観察を基盤として、以下の検証を次サイクルの課題とする：

1. **R₆ が Φ_d[x] を変える構造の解析証明**：なぜ一様場で Φ_d = 0 になるかを代数的に示す。対称性引数による消滅の証明が鍵となる。

2. **3D Φ_d 実装**：現在の 1D ラインサンプリングを 3D 対角方向相関に拡張し、本来の R₆ 演算子能力を評価する。

3. **ノイズ付加テスト**：スキルミオン場に段階的にガウスノイズを加え、Φ_d の SNR 特性と検出閾値を定量化する。

4. **実在材料モデルへの拡張**：LLG 方程式解または実験データへの適用を試み、人工モデルとの定量比較を行う。
