# シュレーディンガーの猫を整数で飼ってみた

> 三角関数もπも使わずに波動関数の位相を一周させたら、誤差がゼロだった。

---

## 猫の話から始める

シュレーディンガーの猫は、箱の中で「生きている」と「死んでいる」を同時に重ね合わせている。

観測するまで、どちらでもある。

私が今日やったことも、だいたいそんな感じです。**「これは物理学なのか」「ただの整数遊びなのか」、観測するまでわからない。**

箱を開けてみましょう。

---

## 通常の波動関数の位相更新

シュレーディンガー方程式が言っていること、ざっくり言うと：

$$\psi(t) = e^{-iEt/\hbar}\,\psi(0)$$

右辺の $e^{i\theta}$ は「単位円上をくるくる回る複素数」です。

実装するとこうなる：

```python
import math

def update_phase_normal(theta):
    return math.cos(theta), math.sin(theta)
```

`math.cos()`、`math.sin()`、浮動小数点、πが登場します。

一周させると：

```python
import math

x, y = 0.0, 0.0
for k in range(360):
    rad = math.radians(k)
    x += math.cos(rad)
    y += math.sin(rad)

print(f"({x:.2e}, {y:.2e})")
# → (-3.41e-14, 8.97e-15)
```

ゼロじゃない。誤差が溜まってる。猫は生きているのか死んでいるのかわからない状態になっています。

---

## B13のやり方：整数で一周する

B13理論（3120B単位系）では、36度刻みの方向ベクトルをあらかじめ整数で定義します。

```python
# BASE = 3120 = 13×8×5×3×2×1×1（フィボナッチ数の積）
# 三角関数は一切使っていない

EDGE_DIRS = [
    ( 3120,     0),   #   0度
    ( 2524,  1834),   #  36度
    (  964,  2967),   #  72度
    ( -964,  2967),   # 108度
    (-2524,  1834),   # 144度
    (-3120,     0),   # 180度
    (-2524, -1834),   # 216度
    ( -964, -2967),   # 252度
    (  964, -2967),   # 288度
    ( 2524, -1834),   # 324度
]
```

`cos(36°) = 2524/3120`、`sin(36°) = 1834/3120`。

近似ではなく、**3120という単位系の定義**です。

位相の更新はこう：

```python
def update_phase_b13(px, py, step):
    dx, dy = EDGE_DIRS[step % 10]
    return px + dx, py + dy
```

足し算だけ。掛け算なし。`import math` すら要りません。

---

## 一周させてみる

```python
x, y = 0, 0
for k in range(10):  # 36度×10 = 360度
    dx, dy = EDGE_DIRS[k]
    x += dx
    y += dy

print(x, y)  # → 0 0
```

**`(0, 0)`。完全にゼロ。**

猫は、ちゃんと生きています（あるいは死んでいます）。重ね合わせが崩れていない。

---

## 何がうれしいのか

| | 通常実装 | B13 |
|---|---|---|
| 位相の表現 | 浮動小数点 | 整数 |
| 使う関数 | cos / sin / π | 足し算のみ |
| 一周の誤差 | 〜10⁻¹⁴ | **ゼロ** |
| テーブル参照速度 | 基準 | **2.8倍速い** |

誤差がゼロなのは「精度が高い」のではなく、**構造的にゼロ**になっています。10ステップの整数の和がちょうど0になるように、3120という単位系が設計されているからです。

---

## 箱はまだ半分閉まっている

これでシュレーディンガー方程式が「解けた」かというと、正直まだわかりません。

位相の回転を整数化できた。誤差がゼロになった。それは確か。

その先は、また別の箱の中にいる猫です。

観測はもう少し待ってください。

## まとめ

```
三角関数ゼロ。掛け算ゼロ。πゼロ。
一周して(0, 0)に戻る。
```

シュレーディンガーの猫は、整数の格子の上で、今日も元気に重ね合わさっています。
