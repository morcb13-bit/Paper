# φ-NTT：加算数論変換

> フィボナッチ系加算格子とリーマンゼータ関数のスペクトル対称性を結ぶ、作用素論的枠組み。

---

## 概要

**φ-NTT（φ加算数論変換）** は、従来の NTT が用いる乗法的巡回群ではなく、線形漸化式（$F_{k+1} = F_k + F_{k-1}$）によって位相生成を行う代替的な変換の枠組みです。

本リポジトリには、φ-NTT の理論的論文と Python 実装が収録されています。離散的な加算格子ダイナミクスと解析的整数論の間に形式的な橋渡しをすることを目的として開発されました。

---

## 主な特徴

- **Carry-free 演算**：$\mathbb{Z}_{10}^B$ 上で桁間の繰り上がりなし
- **黄金比整数環** $\mathbb{Z}[\varphi]$：浮動小数点なし、整数演算のみ
- **B 段テンソル積**：$2^B$ チャネル、段間ツイドル因子ゼロ
- **B = 1〜5 で検証済み**：$\delta$ 畳み込みおよび往復テスト全通過
- **B = 13 閾値**：スペクトル安定化を示唆する構造的観察

## 核心的なアイデア

従来の NTT は乗法的構造に依存します。φ-NTT はこれを**加算再帰核**に置き換えます：

$$K_\phi(n, k) = \omega_N^{\,n \cdot F_k}$$

ここで $F_k$ は $k$ 番目のフィボナッチ数を $N$ で割った余りです。

この枠組みの内部に、反線形対合 $\mathcal{J}$ の不動点集合として正準的な**不変多様体** $\mathcal{M}$ が現れます。$N \to \infty$ における $\mathcal{M}$ の漸近的挙動は、リーマンゼータ関数の臨界線 $\operatorname{Re}(s) = \tfrac{1}{2}$ に形式的に対応すると予想されます。

これはリーマン予想の証明ではなく、**推測的な構造的観察**です。
