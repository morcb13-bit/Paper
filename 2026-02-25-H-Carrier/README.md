# H-Carrier：安定拘束付き残差再注入メカニズム

H-Carrierは、補聴器・空間音響処理・モバイル音声認識などの適応型信号処理系に向けた、軽量な残差再注入メカニズムです。

通常はノイズとして破棄される残差信号を、「安定条件を満たす範囲で」適応バイアスとして再利用します。

---

## 基本概念

各処理ユニットは以下を出力します：

- `H_s` — 符号付き残差  
- `H_m` — 残差強度  

再注入はゲート制御付きで行われます：

g = clip((H_m - τ) / κ, 0, 1)

b = α · g · H_s

安定性を保証する3つの条件：

1. **閾値ゲート** — 定常ノイズを遮断  
2. **リーク積分** — ドリフト防止  
3. **ループ利得拘束** — 有界安定性保証  

G_eff = |α| · g_max · C < 1

---

## 応用例

### 補聴器（カクテルパーティ環境）

- マスク閾値の動的補正  
- 両耳差分残差による空間強調  
- 話者切替時の応答高速化  

### スマートフォン議事録 / ASR

- VAD復帰の高速化  
- ストリーミング閾値の安定化  
- 誤検出の低減  

---

## 設計思想

- フレームあたり O(N) 計算量  
- 追加FFT不要  
- 状態変数はスカラーのみ  
- ハードウェア実装可能  
- Small-Gain条件による明示的安定保証  

---

## コンセプト

残差は常にノイズとは限らない。
安全に扱えば、構造情報のキャリアになり得る。

H-Carrierはそのための安定枠組みを定式化します。
